# Demo 4: MUI-komponenttikirjasto

Tässä demossa on testimielessä toteutettu kaksi versiota:

- Valmis ohjelma, joka toimii suoraan
- Startter-koodi, josta voidaan lähteä [README.md -ohjeistuksella](./starter_code/README.md) rakentamaan sama toteutus.

Startterissa on alustettu Vite-kehitysympäristö ja siivottu kaikki ylimääräinen, jotta ohjelmoinnin voi aloittaa suoraan. Pitää vain muistaa asentaa kaikki kehitysympäristön riippuvuudet ensimmäisellä kerralla suorittamalla komento `npm install`. Kehitysympäristö lähtee käyntiin komennolla `npm run dev`.
